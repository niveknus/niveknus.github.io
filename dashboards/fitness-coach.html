Two-Layer Cloud Architecture
Compute Layer GitHub Actions

Runs garmin_health_snapshot.py + garmin_export_all.py on cron. Handles Garmin OAuth, 2FA, session refresh. Commits daily JSON to raw-sources/garmin/. Traffic routed through dedicated DigitalOcean proxy ($4/mo) to avoid shared-IP 429s.

Intelligence Layer Claude Routine

Fires at 8:30 AM daily. Dispatches fresh Garmin pull, reads digest + profile + rules via GitHub MCP, reasons over recovery state + today's plan, renders polished PNG card, delivers to Telegram via Bot API. Adherence loop at 9:30 PM evaluates workout completion.
